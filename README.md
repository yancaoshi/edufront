# edufront

## Project setup
```
npm install-g @vue/cli
npm install element-plus --save
npm install axios --save

```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
